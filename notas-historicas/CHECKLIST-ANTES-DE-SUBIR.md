# ✅ CHECKLIST ANTES DE SUBIR A APP STORE CONNECT

## 🎯 **URGENTE - CAMBIAR SUBTITLE:**

**ANTES de subir la build, ve a App Store Connect:**

1. Abre: https://appstoreconnect.apple.com
2. Mi App → FinanzasFacil → iOS
3. **App Information** (sidebar izquierdo)
4. Scroll hasta **"Subtitle"**
5. Cambiar de:
   ```
   ❌ "Calculadoras Financieras Gratuitas 2025"
   ```
6. A:
   ```
   ✅ "Calculadoras Financieras"
   ```
7. Click: **"Guardar"** o **"Save"** arriba derecha

---

## ✅ **VERIFICAR:**

- [ ] Subtitle cambiado a "Calculadoras Financieras"
- [ ] App Store Connect guardado
- [ ] Build subida desde Xcode

```
3. Guideline 5.1.1(v): La app NO requiere creación de cuentas para su uso. Todas las funciones principales (calculadoras de salario, hipoteca, autónomos, control de gastos) funcionan sin registro. Las cuentas son OPCIONALES solo para sincronizar datos entre dispositivos. Si un usuario desea eliminar su cuenta, puede contactar: support@finanzasmuyfacil.com

4. Guideline 2.3.7: ✅ Subtitle actualizado a "Calculadoras Financieras" (sin referencias a precios).

Por favor, no probar los botones de autenticación social ya que están temporalmente deshabilitados.

```

## 🎉 **LISTO PARA SUBIR:**

Cuando tengas el subtitle cambiado y las notas copiadas:

1. Ve a: App Store Connect → Mi App → FinanzasFacil
2. Click en la versión 1.0 (o la que estés subiendo)
3. Scroll hasta "Build"
4. Click en "+" junto a "Build"
5. Selecciona tu build recién subida
6. Scroll hasta "Submit for Review"
7. Responde las preguntas de export compliance:
   - **Criptografía:** Selecciona "Does your app use encryption?" → **NO**
8. Pega las notas en "Additional Information"
9. Click: **"Submit for Review"**
